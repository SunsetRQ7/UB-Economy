# UB-Economy Plugin

The UB-Economy plugin is a Minecraft server plugin that manages a diamond-based economy system. It includes support for auctions, player balances, and secure transactions.

## Features
- Manage player diamond balances
- Enable safe deposits and withdrawals
- Auction house functionality
- SQLite database integration

## Requirements
- Java 17+
- Maven 3+

## Getting Started
To build the plugin, use the following Maven command:
```
mvn clean package
```
Place the generated `.jar` file into the `plugins` folder of your Minecraft server.
